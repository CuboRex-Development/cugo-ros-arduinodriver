# Features
CuGo-ROS-Controller-pkgと使うと/cmd_velのベクトルでCuGoの制御を行います。
また、遠隔操作ロボットキット付属のMC-8のプロポを用いて、RC操作と自律走行の切り替えを実施できます。RC-MODEに切り替えることで
無線コントローラで緊急停止と同じ操作ができます。
